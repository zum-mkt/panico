---

# IMPORTANTE

Antes de escrever qualquer linha de código, leia este documento completamente.

Todo o projeto deverá seguir rigorosamente este padrão.

Não improvise.

Não utilize soluções rápidas.

Não utilize templates prontos.

Não copie componentes de outros projetos.

O objetivo é desenvolver um produto premium.

O resultado deve estar acima do padrão encontrado atualmente em empresas funerárias brasileiras.

---

# MISSÃO

Criar um website que transmita:

- acolhimento
- tranquilidade
- confiança
- respeito
- leveza
- elegância
- tradição
- profissionalismo

O visitante nunca deve sentir tristeza.

O visitante deve sentir que existe uma empresa preparada para cuidar de tudo.

---

# POSICIONAMENTO

Este NÃO será apenas um site institucional.

Será uma plataforma institucional completa.

Todo o conteúdo deverá ser administrável.

Todo.

Sem exceção.

# FILOSOFIA

Sempre que existir dúvida entre duas soluções, escolher aquela que:

✔ produz menos código

✔ gera menos manutenção

✔ possui melhor performance

✔ possui melhor acessibilidade

✔ possui melhor UX

✔ possui melhor SEO

✔ possui melhor reutilização
