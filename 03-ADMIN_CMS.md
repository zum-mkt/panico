# 03 - Admin CMS

## URL

/admin

## Objetivo

Controlar 100% do conteúdo do site.

- Estatísticas
- Últimos conteúdos
- Uploads recentes

## Módulos

- Configurações
- Home
- Páginas
- Obituários

- Editor TipTap

- Publicar / Rascunho
- Histórico de alterações
